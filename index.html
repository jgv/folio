Flatfile

(alpha!)

The easiest way to create a portfolio website.

Flatfile aims to be the most simple way for non-technical people to create beautiful portfolio websites. By taking advantage of open directories, Flatfile will transform a simple folder structure into a great looking website for anyone to present their work. Below is a simple example of how this works:

Extra options can be added in the config.php file, found at the root of the project.

Development Goals

  • Detailed installation instructions
  • Individual project pages (coming soon!)
  • Test Suite
  • Configuration options should not be in a .php file
  • Allow for full descriptions of projects
  • Allow for themes
